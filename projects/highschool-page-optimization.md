## 🎯 通用优化模式（参照学院管理）

### 操作按钮统一规范
```
[编辑] 主按钮（蓝色）+ [更多 ▾] 下拉菜单
  ├─ 查看详情
  ├─ 编辑
  ├─ 导出数据
  ├─ 启用/禁用
  └─ 删除（红色，需确认）
```

### 字段扩充思路
每个模块参照高校数字化最佳实践，补充该领域的关键字段。

### 交付流程
后端 Model → Schema → API → 前端 Type → 表格 → 表单 → 详情 → 构建部署
